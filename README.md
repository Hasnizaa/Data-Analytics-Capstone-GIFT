# 📊 B2B Telecom Customer Segmentation & Product Affinity Engine

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Machine Learning](https://img.shields.io/badge/ML-XGBoost%20%7C%20Ensembles-orange.svg)](https://scikit-learn.org/)
[![Tableau Dashboard](https://img.shields.io/badge/Tableau-Interactive%20KPIs-E41F26.svg)](#3-interactive-tableau-dashboard)

An end-to-end data analytics and predictive modeling pipeline designed to decode customer buying behaviors, analyze product category adoption, and mitigate churn risk for a telecommunications provider. This repository transforms over **878,000 transaction line-items** into actionable strategic insights across **4,010 corporate accounts**.
